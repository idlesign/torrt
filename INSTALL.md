# torrt installation

Use `uv` or `pip` to install `torrt`.

## From PyPI

## Install

```shell
uv tool install torrt
```

## Upgrade

```shell
uv tool upgrade torrt
```

## From sources

Use the following command line to install `torrt` from sources directory (containing `pyproject.toml`):

```shell
pip install .
```

or with `uv`:

```shell
uv pip install .
```
